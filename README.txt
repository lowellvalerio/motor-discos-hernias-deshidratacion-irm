MOTOR LINGÜÍSTICO - DISCOS / HERNIAS / DESHIDRATACIÓN V6

Cambios v6:
- Diagnóstico de hernias/protrusiones más natural:
  "Hernia central del disco intervertebral lumbar L5-S1, con desgarro anular posterior y migración caudal, condicionando compromiso radicular bilateral."
- La deshidratación se devuelve como diagnóstico independiente separado por salto de línea.
- En diagnóstico no usa "previamente descrito"; resuelve el nivel real:
  "Deshidratación del disco intervertebral lumbar L5-S1."
- Mantiene endpoint: POST /generar-discos

Render:
Build command: npm install
Start command: npm start

Después de subir a GitHub, en Render usar Manual Deploy > Clear build cache & deploy.
